# React Job Tracker

A frontend project for tracking job applications with React and TypeScript.

## Overview

This project is being built as a business-style application focused on structured data, form handling, CRUD workflows, filtering, sorting, and local persistence.

## Planned Scope

- Create, edit, and delete job applications
- Track application status
- Filter and search records
- Sort by date and status
- Persist data with localStorage
- Display summary metrics in a simple dashboard

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- localStorage

## Status

In progress. Initial scaffold completed.
